# MongoDB Todo App

A backend REST API for managing Todo tasks, built using Node.js, Express.js, and MongoDB Atlas. The application implements JWT authentication, secure user management, and complete CRUD functionality following the MVC architecture.

## Features
- User Registration
- User Login with JWT Authentication
- Create Todo
- Read Todos
- Update Todo
- Delete Todo
- MongoDB Atlas Integration
- Express.js REST API
- MVC Project Structure
- Environment Variable Configuration
